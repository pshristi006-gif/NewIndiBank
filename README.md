# New Indi Bank

A modern banking management website developed using Flask, HTML, CSS and SQLite.

## Features

- Modern Banking UI
- Customer Dashboard
- Deposit & Withdraw
- Login & Signup Pages
- Responsive Design
- SQLite Database

## Technologies Used

- Python Flask
- HTML5
- CSS3
- SQLite
- VS Code

## Run Project

Install Flask:

pip install flask

Run project:

python app.py

Open browser:

http://127.0.0.1:5000
